I wrote this after spending an hour 401-ing on my first Qwen Code prompt because every getting-started guide I read still tells you to log in with a browser - the Qwen OAuth free tier was discontinued on 2026-04-15 and almost nothing online has been updated. The post walks through the three auth paths that actually work now (DashScope, OpenRouter or another OpenAI-compatible provider, or self-hosted vLLM on the open Apache-2.0 weights), plus a six-week side-by-side against Claude Code and Gemini CLI. Curious to hear from anyone running Qwen3-Coder at scale - especially the MoE flagship, where my hardware math broke down.
